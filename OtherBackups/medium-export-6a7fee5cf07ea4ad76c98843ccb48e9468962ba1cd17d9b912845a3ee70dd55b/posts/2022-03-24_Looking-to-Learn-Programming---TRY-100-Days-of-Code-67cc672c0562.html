Looking to Learn Programming — TRY 100 Days of Code

freeCodeCamp and 100 Days of Code are a great way to start your programming journey for free

Recently I came across the 100 Days of Code challenge. The challenge has two parts:

  1. Code at least a little bit each day
  2. Tweet your status (what you did) using the #100DaysOfCode hashtag

I like this challenge because it can be done for free and it keeps you accountable.

  1. freeCodeCamp offers tons of freely available resources in an easy to digest curriculum.
  2. Tweeting everyday keeps you accountable.

I initially saw these as New Years resolution posts. While New Years has come and gone, it is never too late to start pursing a goal! If you want to learn to code, then get started today with the 100 Days of Code challenge.
